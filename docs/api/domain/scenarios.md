# `odys.domain.scenarios`

Scenario definitions for deterministic and stochastic optimization. `Scenario` holds time-series data; `StochasticScenario` extends it with a name and probability.

See [Scenario](../../user_guide/scenario.md) and [Stochastic Optimization](../../user_guide/stochastic.md) in the User Guide.

# `odys.results.optimization_results`

Top-level results container returned by `EnergySystem.optimize()`. Provides access to generator, storage, and market dispatch results, solver status, and objective value.

See [Optimization](../../user_guide/optimization.md) in the User Guide for a full walkthrough.

# `odys.results.dispatch`

Typed dispatch result containers: `GeneratorDispatch` (power, status, startup, shutdown), `StorageDispatch` (net power, SOC, charge mode), and `MarketDispatch` (buy/sell volume). All expose `.to_dataframe()` and `.to_dataset()`.

See [Reading results](../../user_guide/optimization.md#reading-results) in the User Guide.

::: odys.results.dispatch
